# Project-Desk 🎓

A robust, full-stack College Project Allocation and Management System designed to streamline the lifecycle of academic projects. From proposal submission and mentor allocation to weekly progress tracking and documentation, Project-Desk provides an integrated portal for Students, Mentors, and Department Heads.

## 🚀 Features
- **Role-Based Access Control (RBAC):** Dedicated dashboards for Heads, Mentors, and Students.
- **Project Allocation Pipeline:** Seamless submission of project ideas and selection from a Project Bank.
- **Mentor Selection:** Students can request preferred mentors for their projects.
- **Form Management:** Multi-stage project forms (Form-1, Form-2, Form-3) for documentation and evaluation.
- **Progress Tracking:** Weekly evaluation system allowing mentors to grade and provide feedback on student progress.
- **Document Management:** Centralized file repository for project documents and checklist submissions.
- **Announcements:** Real-time messaging system for the Department Head to broadcast notifications.

## 🛠 Tech Stack
- **Frontend:** React, Vite, TailwindCSS, Framer Motion, Axios, Recharts.
- **Backend:** Node.js, Express, Mongoose (MongoDB), Multer (File uploads), ExcelJS/XLSX (Data import).
- **Authentication:** JWT (JSON Web Tokens) with Bcrypt password hashing.

## ⚙️ Installation

### Prerequisites
- Node.js (v18+)
- MongoDB instance
- npm or yarn

### Setup
1. **Clone the repository:**
   ```bash
   git clone https://github.com/KrishAgarwal-44/Project-Desk
   ```

2. **Backend Setup:**
   ```bash
   cd backend
   npm install
   # Create a .env file with MONGO_URI and JWT_SECRET
   npm run dev
   ```

3. **Frontend Setup:**
   ```bash
   cd frontend/Allocation
   npm install
   npm run dev
   ```

## 📁 Project Structure
- `/backend`: Express API, Mongoose models, and controllers for business logic.
- `/frontend/Allocation`: React application with distinct page layouts for different user roles.
- `/backend/uploads`: Storage directory for project documents and checklist files.

## 📖 Usage
- **Head:** Login to manage student/mentor lists via Excel upload, review pending project ideas, and create weekly schedules.
- **Mentor:** Review assigned projects, approve forms, and provide weekly feedback on student tasks.
- **Student:** Submit project ideas or select from the bank, track project milestones, and fill out required documentation forms.
